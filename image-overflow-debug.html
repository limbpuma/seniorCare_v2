Análisis del Problema

❌ PROBLEMA (Código Actual)

w-64 h-64 sm:w-80 sm:h-80 lg:w-96 lg:h-96 xl:w-[28rem] xl:h-[28rem]
  • • xl:w-[28rem] = 448px fijos
  • • No respeta el espacio disponible
  • • Sobresale del contenedor en pantallas estrechas
  • • content-narrow no puede contenerla

✅ SOLUCIÓN

w-full aspect-square max-w-64 sm:max-w-80 lg:max-w-md xl:max-w-lg
  • • w-full: usa el espacio disponible
  • • max-w-*: límites responsivos
  • • aspect-square: mantiene proporción
  • • Se adapta automáticamente al contenedor

📝 Cambios Necesarios

  1. Cambiar tamaños fijos por max-width responsivos
  2. Usar w-full para que la imagen use el espacio disponible
  3. Añadir aspect-square para mantener la proporción
  4. Usar contenedor padre con max-width para control
